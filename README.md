# BMI 健康計算機

這是一個使用 **C# Windows Forms** 製作的 BMI 計算機小程式。  
使用者可以輸入身高與體重，快速計算 BMI，並查看目前體位分類、健康體重範圍與建議文字。介面另外加入了鍵盤操作、腰圍提醒、BMI 公式與跑馬燈，讓整體更完整也更方便操作。:contentReference[oaicite:1]{index=1}

---

## 功能特色

- 輸入 **身高（公分）** 與 **體重（公斤）**
- 自動計算 **BMI 值**
- 顯示 **體位分類**
  - 體重過輕
  - 健康體位
  - 體位過重
  - 輕度肥胖
  - 中度肥胖
  - 重度肥胖
- 顯示 **健康體重範圍**
- 顯示 **建議文字**
- 顯示與理想體重中間值的差距
- 支援 **上下方向鍵切換輸入框**
- 支援 **Enter 直接計算**
- 提供 **清除** 與 **範例** 按鈕
- 顯示 **腰圍提醒**
- 顯示 **BMI 計算公式**
- 右側有 **跑馬燈歡迎訊息** :contentReference[oaicite:2]{index=2}

- Language: **C#**

---

## 操作方式

### 滑鼠操作
1. 在身高欄位輸入身高（公分）
2. 在體重欄位輸入體重（公斤）
3. 按下 **計算**
4. 在結果區查看 BMI、分類、健康體重範圍與建議

### 鍵盤操作
- 按 **↑ / ↓**：在身高與體重輸入框之間切換
- 按 **Enter**：直接計算 BMI
- 按 **Esc**：清除輸入與結果 :contentReference[oaicite:3]{index=3}
